## 🚀 Installation

### 1. StreamElements
- Overlays → New Overlay

### 2. Ajouter widget
- Add Widget → Custom Widget

### 3. Import fichiers
- HTML → `CAM_HTML.html`
- CSS → `CAM_CSS.css`
- JS → `CAM_JS.js`
- Fields → `CAM_FIELDS.json`

### 4. OBS
- Ajouter source navigateur
- Coller URL overlay
